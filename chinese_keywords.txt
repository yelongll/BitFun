空灵 编译器中文关键字映射表
=====================================

基础关键字 (Basic Keywords)
---------------------------
地址 -> addr # 获取变量的内存地址，用于底层内存操作和指针运算
并且 -> and # 逻辑与运算符，用于连接两个逻辑型表达式，当两者都为真时结果为真
作为 -> as # 类型转换操作符，用于安全地将一种类型转换为另一种类型
汇编 -> asm # 内联汇编代码块，允许在空灵代码中嵌入汇编指令进行底层优化
绑定 -> bind # 将标识符绑定到特定符号，常用于重载解析和符号别名
代码块 -> block # 创建代码块，可使用关键字 跳出 ，用于组织代码和异常处理
跳出 -> break # 跳出循环或代码块，立即终止当前循环或关键字 块 的执行
情况 -> case # 多分支条件语句，根据表达式的值选择执行不同的代码分支
强制类型转换 -> cast # 强制类型转换，绕过类型系统进行不安全的类型转换
概念 -> concept # 泛型约束概念，用于定义泛型类型必须满足的条件
常量 -> const # 定义编译时常量，值在编译时确定且不可修改
继续 -> continue # 继续下一次循环迭代，跳过当前循环的剩余部分
转换器 -> converter # 自动类型转换函数，定义类型之间的自动转换规则
无类型 -> untyped
延后执行 -> defer # 延迟执行代码块，在函数退出时执行，无论正常返回还是异常
忽略 -> discard # 显式忽略表达式结果，用于忽略函数返回值或表达式结果
独立类型 -> distinct # 创建不兼容的新类型，即使底层类型相同也不能直接赋值
整除 -> div # 整数除法运算符，返回商的整数部分，舍弃余数
执行 -> do # 代码块标记，常用于循环和条件语句，引入代码块
否则如果 -> elif # 条件语句的"否则如果"分支，当前面的if条件不成立时测试新条件
否则 -> else # 条件语句的"否则"分支，当所有关键字 如果/否则如果 条件都不成立时执行
结束 -> end # 结束代码块或语句，标记代码块的结束
枚举 -> enum # 枚举类型定义，创建一组命名常量的集合
异常 -> except # 异常处理语句，捕获并处理特定类型的异常
导出 -> export # 导出符号到其他模块，使模块内的符号可以被外部访问
最终 -> finally # 异常处理后的清理代码，无论是否发生异常都会执行
对于 -> for # 循环语句，遍历集合中的元素或执行固定次数的循环
选择导入 -> from # 导入特定符号或范围起始，用于选择性导入或指定范围
函数 -> func # 函数定义，创建可能无副作用的函数，适用于函数式编程
如果 -> if # 条件语句，根据逻辑型表达式的值决定是否执行代码块
导入 -> import # 导入模块，将其他模块的符号引入当前命名空间
在 -> in # 成员关系测试或参数传递，检查元素是否在集合中或标记输入参数
包含 -> include # 包含其他文件，将其他文件的内容插入当前位置
接口 -> interface # 接口类型定义，定义一组方法签名，实现多态性
是 -> is # 类型检查运算符，检查对象是否为特定类型或其子类型
不是 -> isnot # 类型检查的否定形式，检查对象是否不是特定类型
迭代器 -> iterator # 自定义迭代器定义，创建可被关键字 对于 循环遍历的自定义数据结构
让 -> let # 不可变变量定义，创建初始化后不能修改的变量
宏定义 -> macro # 宏定义，在编译时生成或转换代码的元编程工具
方法 -> method # 对象方法定义，定义对象的行为，支持动态分派
混入 -> mixin # 模板混入，将模板代码注入到其他代码中
取模 -> mod # 取模运算符，返回整数除法的余数
空 -> nil # 空指针值，表示引用类型变量不指向任何对象
非 -> not # 逻辑非运算符，将逻辑型值取反
不在 -> notin # 成员关系测试的否定形式，检查元素是否不在集合中
对象 -> object # 对象类型定义，创建包含字段和方法的数据结构
当为 -> of # case分支标记或类型转换，用于 情况 语句中的分支或类型转换
或者 -> or # 逻辑或运算符，用于连接两个逻辑型表达式，当至少一个为真时结果为真
输出 -> out # 输出参数标记，标记函数参数为输出参数，用于返回多个值
过程 -> proc # 过程定义，创建可能产生副作用的子程序
指针类型 -> ptr # 指针类型，创建指向内存地址的变量，用于底层内存操作
引发异常 -> raise # 抛出异常，中断正常程序流程并传递错误信息
引用 -> ref # 引用类型，创建指向堆上对象的引用，支持垃圾回收
返回 -> return # 函数返回语句，从函数中返回值并终止函数执行
左移 -> shl # 左移位运算符，将二进制位向左移动指定位数，相当于乘以2的幂
右移 -> shr # 右移位运算符，将二进制位向右移动指定位数，相当于除以2的幂
静态 -> static # 静态变量或方法，与类型关联而非实例关联，或编译时计算
代码模板 -> template # 模板定义，创建在编译时展开的代码模式，实现代码生成
尝试 -> try # 异常处理开始，标记可能抛出异常的代码块
元组 -> tuple # 元组类型，创建包含多个不同类型元素的有序集合
类型 -> type # 类型定义，创建新的类型别名或完全新的类型
使用 -> using # 使用声明，将模块的符号引入当前作用域而无需限定符
变量 -> var # 可变变量定义，创建可以修改值的变量
当 -> when # 编译时条件语句，根据编译时条件决定是否包含代码
循环 -> while # 条件循环，当条件为真时重复执行代码块
异或 -> xor # 异或运算符，对两个操作数的二进制位进行异或运算
生成 -> yield # 生成器返回值，从迭代器中返回值并暂停执行，下次调用时继续



数据类型 (Data Types)
--------------------
整数 -> int # 标准整数类型，通常为32位或64位，取决于平台，用于表示整数
整数8 -> int8 # 8位有符号整数，范围从-128到127，适用于节省内存的小整数
整数16 -> int16 # 16位有符号整数，范围从-32768到32767，适用于中等大小的整数
整数32 -> int32 # 32位有符号整数，范围从-2^31到2^31-1，适用于大多数整数计算
整数64 -> int64 # 64位有符号整数，范围从-2^63到2^63-1，适用于大整数计算
无符号整数 -> uint # 标准无符号整数，通常为32位或64位，只能表示非负整数
无符号整数8 -> uint8 # 8位无符号整数，范围从0到255，常用于字节数据和颜色值
无符号整数16 -> uint16 # 16位无符号整数，范围从0到65535，适用于中等大小的非负整数
无符号整数32 -> uint32 # 32位无符号整数，范围从0到2^32-1，适用于大型非负整数
无符号整数64 -> uint64 # 64位无符号整数，范围从0到2^64-1，适用于超大非负整数
浮点数 -> float # 标准浮点数类型，通常为64位双精度，用于表示实数和小数
浮点数32 -> float32 # 32位单精度浮点数，精度约7位有效数字，适用于节省内存的浮点运算
浮点数64 -> float64 # 64位双精度浮点数，精度约16位有效数字，适用于高精度科学计算
字符串 -> string # 字符串类型，用于存储和操作文本数据，支持Unicode编码
C字符串 -> cstring # C兼容字符串，以关键字 空 结尾的字符数组，用于与C语言库互操作
字符 -> char # 单个字符，存储单个Unicode字符，适用于处理单个文本字符
指针 -> pointer # 通用指针类型，可以指向任何类型的数据，用于底层内存操作
内存地址 -> pointer # 内存地址指针，与关键字 指针 相同，用于存储和操作内存地址
逻辑型 -> bool # 逻辑型类型，只有关键字 真 和 假 个值，用于逻辑判断和条件控制
